# Roadmap - Manual de Sonido LAPEN

## Versión 1.0 - Base de la aplicación

Objetivo: Crear la primera versión funcional.

Incluye:

- Estructura inicial del proyecto
- Manual básico de sonido
- Biblioteca de recursos
- Organización de contenidos
- Diseño inicial de usuarios

---

## Versión 2.0 - Gestión de equipos

Nuevas funciones:

- Registro de usuarios
- Roles de equipo
- Seguimiento de aprendizaje
- Historial de participación

---

## Versión 3.0 - Gestión de eventos

Funciones:

- Crear eventos
- Preparación técnica
- Checklist de montaje
- Control de equipos utilizados
- Reportes de problemas

---

## Versión 4.0 - Expansión técnica

Agregar:

- Iluminación
- Video
- Streaming
- Electricidad
- Logística

---

## Visión final

Convertir la aplicación en una plataforma completa de formación y gestión técnica para equipos de servicio.

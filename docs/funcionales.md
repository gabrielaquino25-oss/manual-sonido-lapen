# Funcionalidades de la App Manual de Sonido LAPEN

## Objetivo

Crear una aplicación que ayude a los equipos de sonido de LAPEN a aprender, organizar y mejorar el trabajo técnico durante eventos.

---

# Módulo 1: Manual de Sonido

## Contenido educativo

- Funcionamiento básico de una consola de sonido
- Tipos de micrófonos y usos
- Conexión de cables y equipos
- Ecualización básica
- Ganancia, volumen y mezcla
- Solución de problemas frecuentes

---

# Módulo 2: Preparación de Eventos

Funciones:

- Crear eventos
- Lista de equipos necesarios
- Checklist antes del evento
- Prueba de sonido
- Registro de problemas encontrados

---

# Módulo 3: Equipo de Trabajo

Funciones:

- Crear usuarios
- Asignar roles:
  - Aprendiz
  - Sonidista
  - Líder técnico
- Seguimiento del aprendizaje

---

# Módulo 4: Biblioteca

Contenido:

- Videos tutoriales
- Documentos PDF
- Guías rápidas
- Material de capacitación

---

# Futuras expansiones

La aplicación podrá crecer para incluir:

- Iluminación
- Video
- Streaming
- Electricidad
- Logística de eventos

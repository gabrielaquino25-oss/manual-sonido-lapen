# 🎛 Manual de Sonido - LAPEN Casuarinas

## Descripción

Manual de Sonido es una aplicación web desarrollada para el área de sonido de LAPEN Casuarinas.

Su objetivo es brindar a los sonidistas acceso rápido a:

- Manuales técnicos
- Fotografías de los equipos
- Videos explicativos
- Preguntas frecuentes
- Solución de problemas
- Recomendaciones de uso

Todo desde cualquier computadora o teléfono móvil.

---

# Objetivos

- Centralizar toda la información técnica.
- Capacitar a nuevos sonidistas.
- Reducir consultas repetitivas.
- Mantener documentación siempre actualizada.
- Escalar el sistema para futuras áreas (Iluminación, Video, Streaming, Electricidad, etc.).

# Roles

## Administrador

- Gestión de usuarios
- Alta/Baja de equipos
- Edición de contenido

- Carga de videos
- Carga de manuales

## Coordinador

- Edición de equipos
- Carga de documentación

## Sonidista

- Consulta de información
- Descarga de manuales
- Visualización de videos
